- **Sales by Body Style** (SUV, Hardtop, Hatchback, etc.)
- **Color Preference Analysis** (Pale White, Black, Red)
- **Regional Dealer Performance** with transmission breakdown
- **Company-wise Sales Trends**
- **Interactive Filters** for dynamic analysis
